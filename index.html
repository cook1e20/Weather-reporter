Local Weather reporter

My second JavaScript, API project a local weather reporter, again this is from FreeCodeCamp and awesome resource to delve into web development.
The projects first challenge was to source the location of the user, to do this I utilised HTML5 Geolocation, this allowed me to get the geographical position of a user although due to privacy issues, the position is not available unless the user allows access. Once the Geolocation was received, in longitude and latitude the next challenge was to utilise this data to access the FreeCodeCamp weather API and return the weather data for that location, and to display this data, for this again I used a JSON request, and displayed the data using JQuery.
